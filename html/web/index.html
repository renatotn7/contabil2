Guia de Estilos - Web

Esta é uma biblioteca de componentes, códigos de UI, padrões de design e guia da marca Bradesco Seguros que permitem que você trabalhe com liberdade para criação, mas ao mesmo tempo protegendo a identidade visual do site Institucional Bradesco Seguros.


A finalidade deste guia é facilitar e oferecer um visual único para todas as páginas criadas por um ou diferentes times de desenvolvimento.

Resolução para aplicações Web



Com o crescente número do uso de dispositivos móveis no Brasil e no mundo, e os grids propostos pelo Bootstrap, a recomendação da arquitetura para projetos web da Bradesco Seguros devem ter no mínimo versões responsivas para as seguintes quebras de layout:


  • ● Desktops (≥992px);

  • ● Tablets (≥768px);

  • ● Celulares (<768px);



*Como ressaltado na conclusão estas são indicações base e deverão ser avaliadas caso a caso, levando em consideração o tempo de desenvolvimento de cada projeto e outros fatores que se façam necessários.
