Grid

A classe "grid" permite que o "elemento" passe a ter o display: flex.

O grid é totalmente responsivo e possui 12 colunas. Para utiliza-las basta adicionar a classe "col" no elemento. As classes indicam o número de colunas que você gostaria de usar das 12 possíveis por linha. Então, se você deseja que o elemento utilize 4 colunas de igual largura, basta usar a classe "col-4".

Caso na classe col não seja definido o número de colunas que o elemento deve ocupar, o elemento ocupará toda a largura restante.

Obs.: É necessário que o elemento pai possua a classe "grid" para que as colunas tenham o devido comportamento.

Classes Alinhamentos

  • Posicionamento das Colunas


  • -wrap - Define o empilhamento das colunas quando necessário automaticamente .

  • -wrap-rev - Define o empilhamento das colunas quando necessário automaticamente em ordem reversa.

  • -row-rev - Os itens são exibidos ao lado na ordem inversa.

  • -columns - Os itens são exibidos verticalmente.

  • -columns-rev - Os itens são exibidos verticalmente na ordem inversa.

  • Alinhamentos Horizontais


  • halign-left - Alinha os itens à esquerda.

  • halign-center - Centraliza os itens.

  • halign-right - Alinha os itens à direita.

  • halign-between - Os itens possuem os mesmos espaçamentos porém encostados nas bordas laterais.

  • halign-around - Os itens possuem os mesmos espaçamentos entre sí, porém centralizados.

  • Alinhamentos Verticais


  • calign-top - Alinha as linhas do container na parte superior.

  • calign-middle - Centraliza as linhas do container verticalmente.

  • calign-bottom - Alinha as linhas do container na parte inferior.

  • valign-top - Alinha os itens na parte superior.

  • valign-middle - Centraliza os itens verticalmente.

  • valign-baseline - Alinha os itens ao conteudo do elemento.

  • valign-bottom - Alinha os itens na parte inferior.

  • self-top - Alinha o item ao superior do elemento pai.

  • self-middle - Alinha o item ao centro.

  • self-bottom - Alinha o item à parte inferior do elemento pai.

  • self-stretch - se o item não tiver um elemento pai (em outras palavras, o item é alinhado baseado no valor padrão ou valor especificado na propriedade align-items, através da classe "valign".

  • self-baseline - Alinha o item baseado no seu conteudo.
